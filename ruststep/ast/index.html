AST (abstract syntax tree) for exchange structure (ISO-10303-21)

Serde data model

Parameter and Record can be deserialize through serde data model.

Parameterserde data model
Integeri64
Realf64
Stringstring
Enumeration-
Listseq
NotProvidedunit
Omittedunit
Typed, Recordmap
RValueenum

See the official document of serde data model for detail.

  • Parameter::Typed e.g. A((1.0, 2.0)) and Record e.g. A(1.0, 2.0) are mapped to “map” in the serde data model
  • Parameter::RValue is mapped to “enum” in the serde data model.
  • FIXME Enumeration is not supported yet.
